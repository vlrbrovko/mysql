# Курсова робота
**Тема:** Клієнт-серверний додаток (UDP + MySQL)
**Виконала:** Бровко В. А., група КБ 6/24

### Команди для запуску:
1. Компіляція сервера: `g++ server.cpp -o server -lpthread -lmysqlclient`
2. Компіляція клієнта: `g++ client.cpp -o client`
3. Запуск: спочатку `./server`, потім `./client`
